# EnzorHtmlParser
Try to get article from site

1) Алгоритм работает тэгами p, span. Авторитетные сайты зачастую кладут текст статьи
   именно в них!

2) Определяет знак препинания в конце. У рекламы редко в конце есть знаак препинания,
   а у статьи почти всегда!

3) Учитывает порядковый номер строки! Тексты статьи расположены близко друг к другу!

4) Учитывает длину строки. Тексты статьи длиннее остальных. Это свойство используестя
   мало, но в будущем можно определить по длинам текста границы статьи, но примерные!
